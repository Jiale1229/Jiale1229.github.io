# 喜新厌旧 uPic (opens new window) 一个自带压缩且简洁的 Mac 图床客户端👍

# 前期准备

  • Github账号：全球最大的代码仓库托管平台，缺陷每个仓库1G容量 (opens new window)
  • PicGo：一个用于快速上传图片并获取图片 URL 链接的工具 (opens new window)
  • Typora：markdown编辑器 (opens new window)
  • Staticaly：轻松地从GitHub / GitLab / Bitbucket等加载您的项目 没有流量限制或限制 (opens new window)

# 创建GitHub仓库

新建一个用来放图片的库，在github首页右上角点击 + ，选择 new reporsitory

# 生成token

创建完成后进入，选择右上角的 Settings

在最下面选择 Developer setting

选择 Personal access tokens ，点击 Generate new token

填写内容

完成创建跳转到生成好的token令牌界面 此处 token只出现一次，一定要另外保存 ，在配置picgo时也需要填入token

# PicGo配置

打开详细窗口

jsdelivr 已挂， 推荐使用 Staticaly

自定义加速域名
https://cdn.staticaly.com/gh/GitHub用户名/仓库名@master
