搭建VuePress个人博客之reco主题

搭建VuePress个人博客之reco主题

🙆‍♂️ 徐乐乐 🗓️ 2022/10/9 🏷️ VuePress

前言

VuePress (opens new window)是一个以 Markdown 为中心的静态网站生成器。可以迅速的将我们的Markdown笔记变成网页，方便我们翻阅，查找。
vuepress-theme-reco (opens new window) 是一款简洁而优雅的 vuepress 博客 & 文档 主题

# 项目搭建

# 1.安装脚手架

# npx
npx @vuepress-reco/theme-cli init

# npm
npm install @vuepress-reco/theme-cli -g

# yarn
yarn global add @vuepress-reco/theme-cli

# 2.项目初始化

# myblog 博客名字自定义
theme-cli init myblog

然后会让输入博客标题、描述、作者，也可以忽略直接回车，待项目创建完成后可在 config.js中配置。 选择项目样式，blog与doc差别不大

  • blog（reco博客）,
  • doc（文档）,
  • 2.x（2.0主题） (opens new window)，

我们用此项目做博客，所以选择blog，回车等待拉取博客模板

# 3.启动项目

# 进到项目根目录中
cd myblog

# 安装依赖包
yarn & npm install

# 等依赖包下完了以后再执行该指令启动项目
yarn dev & npm run dev

# 4.来吧展示 🎉

启动好以后会自动打开网页看到主题的效果。接下来，我们就需要按照 官方 (opens new window) 的一些配置，去修改定制成我们想要的博客。
