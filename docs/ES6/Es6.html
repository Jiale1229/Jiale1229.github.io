ES6 基础特性

ES6 基础特性

🙆‍♂️ 徐乐乐 🗓️ 2022/10/24 🏷️ Es6

特点

两个字：方便。能用更少的代码去做更多的事情，提高工程性。更简短，更整洁，更易读

# let 和 const

与let不同，新的变量声明方式带来了一些不一样的特性，其中最重要的两个特性就是提供了块级作用域与不再具备变量提升。

  • let： 声明局部变量，防止数据污染，只在区块内起作用，外部是不可以调用的。

  • const： 声明常量，可以简单理解为不变的量，从声明开始，这个变量始终不变。

# 箭头函数

# 构成

let 方法名 = 参数 => 函数体;

let f = v => v;
// 等同于
let f = function (v) {
  return v;
};

# 参数

// 没有参数，直接写一个空括号即可。
let f = () => 1;

// 参数只有一个，省去包裹参数的括号。
let f = a => a;

// 有多个参数，将参数依次用逗号分隔，包裹在括号中。
let f = (num1, num2) => num1 + num2;

# 函数体

// 只有一条语句，省去中括号。
let sum = () => 1

// 多条语句，中括号包裹。
let sum = () => {
    const a = 1;
    const b = 2;
    return a + b;
}

// 直接返回一个对象，必须在对象外面加上括号
let getTempItem = id => ({ id: id, name: "Temp" });

// 只有一行语句，且不需要返回值，加一个void关键字，调用一个函数
let sum = () => void doesNotReturn();

# 注意

  1. 函数体内的this对象，就是定义时所在的对象，而不是使用时所在的对象。

  2. 不可以当作构造函数，也就是说，不可以使用new命令，否则会抛出一个错误。

  3. 不可以使用arguments对象，该对象在函数体内不存在。如果要用，可以用 rest 参数代替。

  4. 不可以使用yield命令，因此箭头函数不能用作 Generator 函数。

# 解构赋值

主要分为从 数组 和 对象 中提取值，对变量进行赋值。

如果解构不成功，变量的值就等于undefined。

# 数组

按照位置的对象关系对变量赋值，数组模式和赋值模式统一。 简单理解就是等号的左边和右边相等。

let a=0;
let b=1;
let c=2;

⬇️⬇️⬇️⬇️⬇️⬇️

let [a,b,c]=[0,1,2];
let [a,[b,c],d]=[0,[1,2],3];

# 对象

对象的解构与数组有一个重要的不同。

数组的元素是按次序排列的，变量的取值由它的位置决定；

而对象的属性没有次序，变量必须与属性同名，才能取到正确的值。

let info = {name: 'xjl', sex: '男'}
let {name, sex} = info;

console.log(name) //xjl
console.log(sex) //男

# 字符串

字符串也可以解构，此时字符串被转换成了一个类似数组的对象。

let [a,b,c]="xjl";
console.log(a); // x
console.log(b); // j
console.log(c); // l

# 函数解构

// 解构对象
let info = {name: 'xjl', sex: '男'}
const fn = ({name, sex}) => {
    console.log(name, sex) // xjl 男
}
fn(info)

//解构数组
let num = [1, 2, 3]
const fn = (a, b, c) => {
    console.log(a, b, c) // 1,2,3
}
fn(...num)

# 使用默认值

注意: undefined 和 null 的区别。对象和数组的解构赋值规则一样

let [a, b = "你好"] = ['xjl']
console.log(a) // xjl
console.log(b) // 你好

let [a, b = "你好"] = ['xjl', undefined]
console.log(a) // xjl
console.log(b) // 你好 
// undefined相当于什么都没有，b取默认值

let [a, b = "你好"] = ['xjl', null]
console.log(a) // xjl
console.log(b) // null
// null相当于有值，但值为null。所以b并没有取默认值，而是解构成了null。

# 使用圆括号

如果在解构之前就定义了变量，这时候你再解构会出现问题。 在解构的语句外边加一个圆括号就可以解决

let a;
({a} = {a: '1'});
console.log(a); // 1

# 总结

数组以序列号一一对应，这是一个有序的对应关系。 而对象根据属性名一一对应，这是一个无序的对应关系。 根据这个特性，使用解析结构从对象中获取属性值更加具有可用性。

# 扩展运算符 （…）

… 表示展开运算符，解决参数和对象数组未知情况下的编程

# 参数使用

当写一个方法，传入的参数不确定时。

// 使用...扩展运算符
const sum = (...args) => {
    console.log(args) // [ 1, 2, 3, 4, 5, 6 ]  得到args是一个数组 直接对数组进行操作更加方便
}
sum(1, 2, 3, 4, 5, 6)

// 多个参数时，扩展运算符只能放在最后一个参数
const sum = (a, b, ...args) => { 
    console.log(a) // 1
    console.log(b) // 2
    console.log(args) // [3, 4, 5, 6 ]  
}
sum(1, 2, 3, 4, 5, 6)

# 合并数据

对数组和对象进行合并的行为，是属于浅拷贝

let arr1 = [1, 2, 3]
let arr2 = [...arr1, 4, 5, 6]
console.log(arr2) // [ 1, 2, 3, 4, 5, 6 ]

let name = { name:"xulele" }
let age = { age:18 }
let person = {...name,...age}
console.log(person) // { name: 'xulele', age: 18 }

# 模板字符串

模板字符串（template string）是增强版的字符串，用 反引号 ` 标识。

它可以当作普通字符串使用，也可以用来定义多行字符串，或者在字符串中用 ${} 将变量括起来，解决字符串拼接问题。

# ` 定义字符串

注意

如果使用模版字符串表示多行字符串，所有的 空格 和 缩进 都会被保存在输出中！！

// 普通字符串
`string text`

// 多行字符串
`string text line 1
 string text line 2`

// 在字符串中使用反引号 前面要用反斜杠转义
`\`Yo\` World!` // `Yo` World!

# ${} 引用

// 引用变量
let name = 'xulele', age = 18;
let desc = `${name}今年${age}岁了`

// 引用表达式
`今年${10+8}岁` // 今年18岁

// 引用方法
const age = () => 10 + 8
console.log(`今年${age()}岁`) // 今年18岁

# 对象字面量

  • 计算属性名：属性名来自于某个表达式的值，可以用中括号来表示该属性名
  • 方法简写：省略冒号和function关键字
  • 属性简写：属性名和变量名同名，可以只写变量名
let name = 'xulele',
    age = 18,
    prop = 'home';

let person = {
    [prop]: '河南',
    [prop + 'City']: '郑州', // 计算属性名
    name,  //属性简写
    age,
    bio() { //方法简写
        console.log(`大家好，我是${this.name}，今年${this.age}岁。`);
    }
}
console.log(person)  

// {
//     home: '河南',
//     homeCity: '郑州',
//     name: 'xulele',
//     age: 18,
//     bio: [Function: bio]
// }

person.bio() // 大家好，我是xulele，今年18岁。
