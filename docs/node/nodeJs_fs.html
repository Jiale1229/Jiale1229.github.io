fs 文件模块

fs 文件模块

🙆‍♂️ 徐乐乐 🗓️ 2022/10/21 🏷️ NodeJs

概述

Fs模块专门用来操作文件和目录。能够进行文件创建/删除，读取文件，获取文件详情等操作。

# 导入模块(fs)

const fs = require("fs")

# 同步异步

模块中的方法均有异步和同步版本，异步的方法函数最后一个参数为回调函数，回调函数的第一个参数包含了错误信息(error)。

建议使用异步方法，比起同步，异步方法性能更高，速度更快，而且没有阻塞。

fs.readFile()  //异步
fs.readFileSync()  //同步

# 常用方法

# fs.readFile()-->读取文件

获取的内容默认是Buffer数据，调用toString方法转为字符串之后查看

fs.readFile('/demo.text',(err,data)=>{
    if (err) {
        return console.error(err);
    }
    console.log(data.toString());
})

# fs.readdir()-->读取目录

获取的文件列表 files 为数组形式

fs.readdir('/images/',(err,files)=>{
    if (err) {
        return console.error(err);
    }
    console.log(files);
})

# fs.mkdir()-->创建目录

recursive 默认为 false。 以递归的方式创建目录

fs.mkdir('/picname/', { recursive: true }, err => {
  if (err) throw err;
});

# fs.rename() -->移动图片

可更改文件存放路径，也可修改文件名称。

fs.rename('/oldPath/','/newPath/', err => {
    if (err) throw err;
})

# fs.writeFile() -->写入文件

如果文件存在，该方法写入的内容会覆盖旧的文件内容。

fs.writeFile('test.html', 'Hello World', err => {
    if (err) throw err;
})
